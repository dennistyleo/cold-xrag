# Cold-XRAG User Guide

## Introduction
Cold-XRAG is an advanced data exploration and signal analysis dashboard combining robust statistical signal characterization (L1-L5 pipelines) with an interactive 3D Geo/World visualization engine.

## Usage
1. **Domains**: Select a domain on the left.
2. **Categories**: Selecting a domain populates specific analysis categories.
3. **Run Analysis**: Clicking **"Start Analysis Pipeline"** will stream through L1 (Stationarity, SNR), L2 (Candidates), L3 (Feasibility), L4 (Calibration), and L5 (Explainability), concluding with a success readout and audit hash.
4. **Geo & World 3D Tabs**: Explore Graph Neural Network properties via draggable, interactive 3D visualizations.
5. **Charts**: Observe RSI, EMA, Core Analysis, and Causal Relationships updating in real-time. Use the formula search filter to find axioms quickly.
